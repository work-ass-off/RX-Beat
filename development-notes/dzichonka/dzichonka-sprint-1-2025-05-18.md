### Track Card Component

Компонент принимает данные через input:
`track = input.required<JamendoTrack>();`

При клике на карточку происходит установка текущего трека в глобальный стор:

```
private trackStoreService = inject(PlayerStoreService);

onClick(): void {
  this.trackStoreService.setTrack(this.track());
}
```

Добавила обработчик события через host binding:
`host: { '(mousedown)': 'onClick()' }`

### Player Component

Компонент плеера отображается в футере и реагирует на выбор трека.

```
private trackStoreService = inject(PlayerStoreService);
track = this.trackStoreService.currentTrack;
```
